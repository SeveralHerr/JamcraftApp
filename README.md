# Jamcraft

Personal portfolio and community hub for James Herr - Full Stack Engineer & Game Developer.

## What is this?

A website that serves as a central hub connecting to my various platforms. 

## Quick Start

```bash
cd jamcraft-app
npm install
npm run dev
```

Visit `http://localhost:5173` to see the site.

## Tech Stack

- React 19 + TypeScript
- Vite (build tool)
- Mantine UI (component library)
- React Router (navigation)

## Project Philosophy

This codebase attempts to follow **Clean Architecture** and **Screaming Architecture** principles.


## Scripts

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run preview  # Preview production build
npm run lint     # Run ESLint
```
