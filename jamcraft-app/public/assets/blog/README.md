# Blog Assets

This directory contains images and other assets used in blog posts.

## Adding the Calendar Refactoring Screenshot

For the blog post "Calendar Refactoring for Better Collaboration" (2025-01-27), you need to:

1. Download the screenshot from the original LinkedIn post:
   https://www.linkedin.com/posts/james-herr-63b85b1b3_mobprogramming-ensembleprogramming-refactoring-activity-7289760836452028416-7JFK

2. Save it as: `calendar-refactoring-2025-01-27.png`

3. Place it in this directory (`public/assets/blog/`)

The blog post markdown already references this image at:
```markdown
![Calendar Refactoring Structure](/assets/blog/calendar-refactoring-2025-01-27.png)
```

## File Naming Convention

Use descriptive names with dates for blog images:
- Format: `[topic]-YYYY-MM-DD.[extension]`
- Examples:
  - `calendar-refactoring-2025-01-27.png`
  - `code-review-tips-2024-09-15.jpg`
  - `deployment-workflow-2025-02-10.png`
